P02: Style Guide

Default Button

CSS selector:

.button

Sample code:

<a class="button" href="#">INPUT</a>

Rendered element:

INPUT

In-text link

CSS selector:

.text-link

Sample code:

<p>Here is an <a href="#" class="text-link">in-text link</a>, the link is over</p>

Rendered element:

Here is an in-text link, the link is over.
